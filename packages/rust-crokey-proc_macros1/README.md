# rust-crokey-proc_macros1

Compatibility package for the published `crokey-proc_macros 1.3.0` release
required exactly by `crokey 1.3.0`. The proc-macro crate omits the workspace
license file, so the package carries the exact MIT text shipped by the sibling
published `crokey 1.3.0` crate. Clean Fedora 43 and Fedora 44 x86_64 mock builds
passed, and the package is enabled.
