# rust-lazy-regex3

Compatibility package for the published `lazy-regex 3.4.1` release pinned by
the `termimad 0.34.1` lockfile. Only the default and implicit `regex` features
required by this dependency chain are exposed initially. The package remains
Clean Fedora 43 and Fedora 44 x86_64 mock builds passed, and the package is
enabled.
