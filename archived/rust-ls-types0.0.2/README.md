# rust-ls-types0.0.2

Compatibility package for the published `ls-types 0.0.2` release pinned by
the `tower-lsp-server 0.23.0` lockfile. Later `0.0.x` releases have an
incompatible structure API, so update automation retains this exact release.
Generated with rust2rpm 28. Clean Fedora 43 and Fedora 44 x86_64 mock builds
passed, and the package is enabled.
